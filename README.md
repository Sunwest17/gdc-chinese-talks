# GDC 有字幕视频中文报告

纯 HTML/CSS/JS 静态站，可部署到 GitHub Pages。首页为 `index.html`，数据在 `data.js`。

## 本地预览

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 内容来源

当前批次只收录能取得 YouTube 字幕或转写文本的 GDC 官方视频、演讲片段和 keynote。GDC Vault 只有简介、没有原文或字幕的链接不进入页面；YouTube 字幕禁用、抓取失败、非演讲内容的链接也会剔除。
